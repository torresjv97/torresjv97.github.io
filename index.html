Raycasting Demo

A small prototype written in Javascript to visualize and learn the theory of how 3D rendering was implemented in Wolfenstein 3D with Raycasting, which is a rendering technique used to create a 3D perspective in a 2D map. Back when computers were slower it wasn't possible to run real 3D engines in realtime, and raycasting was the first solution. Raycasting can go very fast, because only a calculation has to be done for every vertical line of the screen.

Doodle Dodge

A physics based puzzle game made for mobile (Android). In this game you try to protect the ball (player), from various hazards such as saw and spikes. To do this, you need to draw a route for your ball. But you need to be strategic, as this path/trail fades after time. In this game I: Implemented the base gameplay and mechanics, Designed the levels with a level selection along with an "Endurance Mode", implemented A level progression system to unlock levels after completion, and Integrated Unity Ads.
